Shoot-The-Words

A simple educational game that helps toddlers practice breaking words into syllables or sounds to improve pronunciation and general foundations for reading.

How It Works

The game shows three guns. Child taps one of the guns and identifies the order of the broken up words. Positive reinforcement for correct answers. If wrong, just reset and try again!

How to Run

Clone the folder, or download it.

Requires Python 3.x

Requires pyttsx3 for sound and tkinter/powershell on windows.

Current audio path is windows-friendly, additional changes may be needed for non-windows users.
